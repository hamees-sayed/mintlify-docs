The **Lightning Large API** provides enhanced text-to-speech capabilities with support for advanced configurations and voice customization. Below are the endpoints and their details.

## Endpoints

### 1. Get Speech
Convert text into natural-sounding speech using the **Lightning Large** model.

#### **Request URL**
```
https://waves-api.smallest.ai/api/v1/lightning-large/get_speech
```

#### **Request Method**
`POST`

#### **Headers**
- `Authorization: Bearer <your-api-key>`
- `Content-Type: application/json`

#### **Payload**
| Parameter         | Type      | Required | Description                                          |
|-------------------|-----------|----------|------------------------------------------------------|
| `text`            | `string`  | Yes      | The input text to convert to speech.                |
| `voice_id`        | `string`  | Yes      | The ID of the voice to use for synthesis.           |
| `speed`           | `float`   | No       | Adjust playback speed (default: `1.0`).             |
| `add_wav_header`  | `boolean` | No       | Adds a WAV header to the output file (default: `false`). |
| `cfg`             | `object`  | No       | Configuration for advanced settings (default: `null`). |

#### **Example Request**

```bash
curl --location 'https://waves-api.smallest.ai/api/v1/lightning-large/get_speech' \
--header 'Authorization: Bearer <your-api-key>' \
--header 'Content-Type: application/json' \
--data '{
    "text": "Hey there",
    "voice_id": "blofeld",
    "speed": 1.0,
    "add_wav_header": true,
    "cfg": 2.0
}'
```

### 2. Create Voice
Upload a voice file to create a custom voice profile.

#### **Request URL**
```
https://dev-waves-api.smallest.ai/api/v1/lightning-large/create
```

#### **Request Method**
`POST`

#### **Headers**
- `Authorization: Bearer <your-api-key>`

#### **Payload**
| Parameter      | Type     | Required | Description                       |
|----------------|----------|----------|-----------------------------------|
| `file`         | `file`   | Yes      | The voice file to upload.         |
| `displayName`  | `string` | Yes      | The display name for the custom voice. |

#### **Example Request**

```bash
curl --location 'https://dev-waves-api.smallest.ai/api/v1/lightning-large/create' \
--header 'Authorization: Bearer <your-api-key>' \
--form 'file=@"/path/to/voice-file.wav"' \
--form 'displayName="customVoiceName"'
```
